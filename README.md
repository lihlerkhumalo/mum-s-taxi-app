# 🚖 Mum’s Taxi Web App

## 📌 Description
This web application simulates a taxi service platform, allowing users to interact with transport-related features. It focuses on user interface design and mobile-first responsiveness.

## 🚀 Features
- Interactive user interface
- Mobile-first responsive design
- Structured layout for easy navigation
- User-friendly experience

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 What I Learned
- Responsive web design principles
- UI/UX design basics
- Structuring front-end applications
- Writing modular and maintainable code
